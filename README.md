# Anime or Sage Game
## A web-based game to test your knowledge of quote origins
### Can you guess if the inspirational quote comes from an anime character or a real-life leader?

#### Technology Used
- Bootstrap CSS
- API Ninja quote API
- Anime API
- Vanilla HTML/CSS/JavaScript
